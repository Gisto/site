author: Sasha Khamkov

---

Gisto is a code snippet manager that integrates with GitHub, GitLab snippets and loacal storage option, adds additional features such as searching, tagging and sharing snippets while including a rich code editor.

All your data is stored with your chosen provider(s) and you can access it via provider platforms (e.g., GitHub or GitLab) with changes carrying over to Gisto.
