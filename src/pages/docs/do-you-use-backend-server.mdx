title: Do you use backend server?
author: Sasha Khamkov
category: 2. F.A.Q.
published: 2005-02-16
---

Gisto primarily functions as a client-side application that interacts with GitHub Gists via the GitHub API.

This means it relies on GitHub's back-end for storing and managing gists, rather than having its own dedicated server-side infrastructure.

Gisto store nothing except of its own configurations.
