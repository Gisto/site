## Overview

Tags in Gisto help organize and categorize your gists for easier management and retrieval. This guide explains how to create and manage tags in Gisto.

## Creating Tags

Select an existing gist or create a new one.

Anywhere in the description field create a tag word with a prefix of `#`, e.g. to create tag "temporary" - use `#temporary`

## Managing Tags

Removing a Tag: Click the "X" next to a tag to remove it.

Editing a Tag: Remove the existing tag and optionally add a new one.

Searching by Tags: Use the search bar to find gists associated with specific tags.
