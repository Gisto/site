title: Do you use backend server?
author: Sasha Khamkov
category: 2. F.A.Q.
published: 2025-02-16
---

Gisto primarily functions as a client-side application that interacts with GitHub and GitLab snippet APIs or local in-browser database.

This means it relies on the providers' back-ends for storing and managing snippets, rather than having its own dedicated server-side infrastructure.

Gisto stores nothing except its own configurations.
