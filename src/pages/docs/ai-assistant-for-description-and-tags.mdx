## Overview

Gisto integrates with several AI assistants which can help you generate descriptions and tags for your snippets. This feature is designed to enhance your workflow by providing intelligent suggestions based on the content of your code snippets.

## Get your API key

To use the AI assistant, you need to get an API key from desierred provider. Follow the steps in the settings.

## Add API key

To use the AI assistant, you need to add your API key. Follow these steps:

1. Open Gisto and navigate to the settings page.
2. Find the section with AI assistant.
3. Enter your API key in the provided field.

## Use the assistant

1. Edit snippet or create a new one.
2. Add your code snippet content and file names.
3. Click on the "AI" button located in the description field. **This button will only be visible if you have added API key**.
4. The AI assistant will analyze the content of your code snippet and generate a description and tags.
5. Review the suggestions and make any necessary adjustments.

The generated description and tags will be saved with your snippet, making it easier to find and manage in the future.
